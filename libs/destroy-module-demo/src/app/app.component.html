DestroyModule demo

Click <Route to AModule> link to navigate to lazy loading `AModule`. Module is providing `LoopService`. Service emits console log events periodically. And this module and service won't be destroyed. `AModule` imports `DModule` and it won't be destroyed to.

Click <Route to B-CModule> link to navigate to lazy loading `BModule` and sub `CModule`. Each of modules is providing same `LoopService`. Service emits console log events periodically as at the `AModule`. And this modules and provided services will be destroyed with back navigation also will be destroyed imported `DModule`.

Please open console to see 'LoopService' console log events emission from each alive module.
